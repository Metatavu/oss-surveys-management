### OSS Surveys Management UI

This project uses Vite.

##### Commands

Install packages

```
npm i
```

Run project with

```
npm run dev
```

Build project

```
npm run build
```

Preview production build locally

```
npm run preview
```

Check linting and formatting

```
npm run lint:check
```

Automatically fix auto-fixable linting errors and format all files

```
npm run lint:fix
```
